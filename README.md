## 🧠 LLM & Prompt Engineering

**What is an LLM?**  
Large Language Models (LLMs) are transformer-based neural networks trained on massive text corpora. They predict tokens using attention mechanisms across billions of parameters, enabling tasks like reasoning, summarisation, code generation, and conversation.

**Prompting techniques I apply:**
- **Role-based prompting** — `"You are a senior data analyst. Analyse this dataset and..."`
- **Instruction-based prompting** — explicit step-by-step directives for structured outputs
- **Few-shot prompting** — providing examples to guide output format
- **Chain-of-thought** — asking the model to reason step by step before answering
- **Output format constraints** — specifying JSON, bullet points, or table formats to make outputs usable in pipelines

**Example prompt pattern:**
```
You are a data analyst. Given the following CSV data, identify the top 3 signals 
predicting customer churn. Return your answer as a JSON object only, no explanation.
```
